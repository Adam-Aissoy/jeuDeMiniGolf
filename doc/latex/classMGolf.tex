\doxysubsection{Detailed Description}
La classe \mbox{\hyperlink{classMGolf}{M\+Golf}} contient \+: 


\begin{DoxyItemize}
\item Une window, un renderer
\item Toutes les images nécessaires à l\textquotesingle{}affichage
\item Le delta Time
\item 1 \mbox{\hyperlink{classGame}{Game}}
\item Les entiers correspondant à la taille de la fenêtre
\item Une variable state (0 pour title Screen 1 pour Play)
\item Un booléen M\+Pressed correspondant au clic gauche de la souris
\item Des S\+D\+L\+\_\+\+Point pour les différentes positions de la souris
\item Des S\+D\+L\+\_\+\+Rect pour pouvoir clicker sur des images.
\item Un booléen quit qui reste false tant que l\textquotesingle{}on n\textquotesingle{}a pas quitté la fenêtre 
\end{DoxyItemize}
